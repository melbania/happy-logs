Origins

Learning Logs takes its name (and most of its structure and functionality) from the web application tutorial found in Chapters 18 - 20 in the beginner's python programming book Python Crash Course by Eric Matthes.

Some modifications have been made to the original design at the request of potential users (me, my son, and the hubs):

  • used a fixed-top navbar instead of a static-top navbar

  • added some more bootstrap3 buttons

  • added this "about" page with panels

  • sorted user's topics alphabetically instead of by date added: required learning more about Django's QuerySet API and the ordered_by method

  • added the image of a happy log on the index page: original image found on March 11, 2016 on www.kingofwallpapers.com via a google search for images of logs; modified by me using Adobe Photoshop CS5.

  • added timezone aware dates for entry views rendered client-side on the "topic" page: required some javascript code (suggested by the hubs) and installation of moment.js (an amazing little package that I found which provides formatting options for javascript date objects! ... required downloading and using moment.min.js as a static file)

  • used a markdown editor/viewer for the "new entry", "edit entry", and "topic" pages: required installation of the projects django-pagedown (for the markdown editing) and django-markdown-deux (for viewing markdown entries ... and for safely escaping raw HTML)

How to Use Learning Logs

  1. Register for your free account.

    2. Click on the "register" link found on the right side of the navigation bar at the top of the page.


    Create a unique username and password, then click on the "register" button at the bottom of the form.


  2. Go do some learning about stuff ... maybe things.


  3. Come back to Learning Logs and Log in to your account.

    4. Click on the "log in " link found on the right side of the navigation bar at the top of the page.


  4. Add a topic.

    5. Click on the "Topics" tab found toward the middle of the navigation bar at the top of the page.


    Click on the "add a topic" button at the bottom of the topics list panel.


    Type the new topic's name in the form. Then click the "add topic" button at the bottom of the form to save that topic.


  5. Add an entry for that topic.

    6. Click on the topic's name.


    Click the "add an entry" button to go to the text editor.


    In the text editor pane, type up your entry. Scroll down to see a preview of that entry in the preview pane. Click the "save entry" button when done.


  6. Edit an entry if necessary.

    7. Click on the "edit entry" link found in the entry's time and date header.


    Dress that entry up with some Markdown if you're feeling fancy. Click the "save changes" button when done.


  7. Rinse and repeat.

About Markdown

Markdown is like shorthand for HTML. It can do a lot of what HTML does to text, but with fewer keystrokes. I chose to include it as a feature of Learning Logs because it provides the ability to add meaning and readability to what would otherwise be plain text entries.

Click here to learn lots more about Markdown.

Click on the "Markdown Cheatsheet" link in the middle of the navigation bar at the top of the page for help with common syntax. The page opens in a new tab.
